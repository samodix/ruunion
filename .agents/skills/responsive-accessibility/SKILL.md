# Responsive Accessibility

## Responsive

- Mobile first.
- Tester au minimum 390px.
- Aucun débordement horizontal.
- Grilles fluides.
- Images non déformées.
- CTA utilisables au doigt.

## Accessibilité

- Contraste suffisant.
- Focus visible.
- Navigation clavier.
- `alt` pertinent sur images informatives.
- `aria-label` sur icônes/liens ambigus.
- Labels de formulaires.
- Messages d’erreur compréhensibles.
- Respect de `prefers-reduced-motion`.

## WordPress/WooCommerce

Ne pas casser les hooks, labels, notices, champs checkout, panier ou liens de récupération de mot de passe.
