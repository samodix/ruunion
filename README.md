# RU Union

Socle de la plateforme RU Union construit avec Next.js, React, TypeScript et Tailwind CSS.

## Démarrer

```bash
npm install
npm run dev
```

Le site est ensuite disponible sur `http://localhost:3000`.

## Vérifications

```bash
npm run lint
npm run build
```

Les contenus et services externes sont mockés. Aucun appel WordPress, WooCommerce, Stripe ou PayPal n’est effectué dans cette version.

Consultez le dossier `docs/` pour le plan, les lots, la stratégie API et l’environnement.
